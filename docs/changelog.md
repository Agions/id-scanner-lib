# 更新日志

所有重要的版本变更都会记录在此文件中。

## [1.5.0] - 2025-06-05

### 新增

- **OCR处理与防伪检测增强**：增强ID卡文本解析准确率，实现UV墨水和微文本的实际检测算法
- **类型定义完善**：新增browser-image-compression和tesseract.js库的类型定义，提升开发体验

### 改进

- **CI/CD流程**：修复GitHub Actions工作流配置，提升持续集成与部署的稳定性
- **项目结构**：优化项目依赖管理，移除package-lock.json文件，简化安装流程

## [1.4.0] 

### 新增

- **模块化架构重构**：完全重构项目为模块化架构，提高可维护性和扩展性
- **二维码扫描**：新增对QR码和多种条形码格式的支持
- **活体检测增强**：添加主动式活体检测模式，支持眨眼、点头等动作验证
- **综合演示**：新增综合演示页面，展示多功能同时使用
- **脚手架工具**：添加命令行脚手架工具，快速生成模板代码
- **API文档**：完善详细的API文档和使用示例

### 改进

- **性能优化**：优化模型加载和处理逻辑，减少资源占用
- **错误处理**：实现层次化的错误处理系统
- **事件系统**：重写事件发布订阅系统，支持更灵活的事件处理
- **资源管理**：优化资源加载和释放机制
- **摄像头管理**：重构摄像头管理模块，支持更多配置选项

### 修复

- 修复在某些浏览器下摄像头无法正确初始化的问题
- 修复人脸检测精度下降的问题
- 修复内存泄漏问题

## [1.3.3]
